# GT Walsheim Font Files

Поместите файлы шрифта GT Walsheim Medium в эту папку:

- `GT-Walsheim-Medium.woff2` (предпочтительно)
- `GT-Walsheim-Medium.woff` (fallback)
- `GT-Walsheim-Medium.ttf` (fallback)

## Где получить шрифт:

GT Walsheim - это коммерческий шрифт от Grilli Type.
Вы можете приобрести его на: https://www.grillitype.com/typefaces/gt-walsheim

После покупки скачайте файлы шрифта и поместите их в эту папку.

## Альтернативный способ (CDN):

Если у вас есть доступ к шрифту через CDN или другой сервис (например, Adobe Fonts),
вы можете обновить `@font-face` в `src/index.css` для использования URL вместо локальных файлов.
